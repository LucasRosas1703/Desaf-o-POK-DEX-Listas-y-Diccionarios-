# Pokédex

Este programa te permite ingresar Un pokemon con su nombre, tipo, nivel y poder, y luego mostrar todos tus pokémones en una lista

## Funcionalidades (CRUD)
Este proyecto incluye las siguientes funcionalidades a través de un menú interactivo:
- **Create**: Permite agregar un nuevo [elemento] al sistema.
- **Read**: Muestra la lista completa de [elementos] registrados.
- **Update**: Modifica los datos de un [elemento] existente buscando por su nombre/ID.
- **Delete**: Elimina un [elemento] del sistema.

## Requisitos Técnicos Aplicados
- Uso de listas y diccionarios.
- Uso de funciones documentadas.
- Validaciones y manejo de excepciones (`try/except`).
